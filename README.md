# osm-navigator

Open-source mapping and navigation SDK for React Native (bare and Expo). This monorepo is scaffolded for:
- Map rendering (MapLibre Native via Expo Module)
- Turn-by-turn routing (Valhalla REST API)
- Geocoding (Photon REST API)
- OpenFreeMap/Protomaps tile sources

## Packages
- @osm-navigator/core
- @osm-navigator/native-map
- @osm-navigator/ui-navigation
- apps/example (demo app)

## Requirements
- Yarn Classic (v1) workspaces
- TypeScript strict mode, ES2020
- Expo SDK 51+, React Native 0.74+ (New Architecture)
- No proprietary dependencies

## Getting Started
1. Install dependencies: `yarn install`
2. See each package README for usage
