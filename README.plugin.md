# Expo Config Plugin: osm-navigator

This plugin integrates the native MapLibre module and configures permissions for the OSM Navigator SDK.

## Usage
Add to your app.json or app.config.js:

```js
module.exports = {
  plugins: [
    ["../app.plugin.js"]
  ]
};
```

## What it does
- (Planned) Links MapLibre native module
- (Planned) Adds required permissions
- (Planned) Configures custom props for MapView
